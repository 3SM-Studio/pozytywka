# Pozytywka

Monorepo strony internetowej Pracowni Twórczej Pozytywka.

## Stack

- Next.js
- React
- TypeScript
- Sanity Studio
- pnpm workspaces
- Turborepo

## Struktura

- `apps/web` — publiczna strona internetowa
- `apps/studio` — Sanity Studio
- `packages/eslint-config` — wspólna konfiguracja ESLint
- `packages/typescript-config` — wspólna konfiguracja TypeScript

## Status

Monorepo zawiera aplikację Next.js oraz Sanity Studio dla datasetu `development`.
